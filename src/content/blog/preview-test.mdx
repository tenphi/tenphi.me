## Interactive JS preview (lazy srcdoc iframe)

This one loads on scroll and runs in an isolated, null-origin sandbox.

<Preview
  title="Counter"
  html={`<button class="counter">Clicked 0 times</button>`}
  css={`
    .counter {
      padding: 0.5rem 1rem;
      border: 1px solid var(--border-color);
      border-radius: var(--radius);
      background: var(--surface-2-color);
      color: var(--text-color);
      font: inherit;
      cursor: pointer;
    }
  `}
  js={`const b = document.querySelector('.counter');
let n = 0;
b.addEventListener('click', () => {
  n += 1;
  b.textContent = 'Clicked ' + n + ' times';
});`}
/>
