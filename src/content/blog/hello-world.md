Welcome to my blog. I'm Andrey Yamanov — a frontend engineer who enjoys building tools for other developers. This is a fresh start, and I wanted to take a moment to explain what this space is about.

## The stack

This site is built with [Astro](https://astro.build/) and styled with [Tasty](https://github.com/tenphi/tasty), a CSS-in-JS system I've been working on. The color scheme is generated by [Glaze](https://github.com/tenphi/glaze), an OKHSL-based color theme generator that produces WCAG-compliant light and dark palettes from a single hue. The whole thing compiles to static HTML and CSS — zero JavaScript in the browser.

## What to expect

I plan to write about frontend architecture, design systems, CSS engineering, and the tools I build. Posts will be infrequent but (hopefully) worth reading. If you're interested in how styling systems work under the hood, or how to build component APIs that scale, this is the right place.

Stay tuned.
